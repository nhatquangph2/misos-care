## üòä Emotional States

Miso needs 5-7 core emotional expressions:

### 1. Happy / Welcoming
**Use:** Default state, homepage, successful actions
**Expression:** Big smile, bright eyes, upward tail
**Pose:** Waving fin, welcoming gesture

### 2. Supportive / Caring
**Use:** During difficult test results, encouragement
**Expression:** Gentle smile, warm eyes, hand/fin on heart
**Pose:** Hugging gesture, open arms

### 3. Excited / Celebrating
**Use:** Test completion, achievements, milestones
**Expression:** Wide smile, sparkly eyes, confetti
**Pose:** Jumping, arms up, party hat optional

### 4. Calm / Meditative
**Use:** Stress management, mindfulness exercises
**Expression:** Peaceful, closed eyes, serene smile
**Pose:** Meditation pose, lotus position

### 5. Thinking / Curious
**Use:** During test questions, exploring
**Expression:** Slight smile, one eye squinting, thinking
**Pose:** Fin on chin, question mark nearby

### 6. Encouraging / Strong
**Use:** Building resilience, goal setting
**Expression:** Determined smile, confident eyes
**Pose:** Flexing, superhero pose, cape optional

### 7. Resting / Sleepy
**Use:** Daily mood check-in (tired state), bedtime reminders
**Expression:** Droopy eyes, small smile, yawning
**Pose:** Laying down, pillow, nightcap

## üìê Technical Specifications

### File Formats
- **Primary:** SVG (scalable, small file size)
- **Fallback:** PNG @2x, @3x for retina
- **Animation:** Lottie JSON (for complex animations)

### Dimensions
- **Icon size:** 64x64px, 128x128px, 256x256px
- **Hero size:** 800x800px minimum
- **Illustration size:** 1200x1200px for detail

### Color Modes
- **Light mode:** Bright Miso blue (#3b82f6)
- **Dark mode:** Lighter Miso blue (#60a5fa)
- Ensure good contrast in both modes

### Accessibility
- Alt text for all character images
- Decorative vs meaningful (use alt="" for decorative)
- Ensure character doesn't block important content

## üí° Usage Examples

### React Component
```tsx
import Image from 'next/image'

export function MisoCharacter({
  emotion = 'happy',
  size = 'md'
}: {
  emotion: 'happy' | 'supportive' | 'celebrating' | 'calm' | 'thinking' | 'strong' | 'sleepy'
  size: 'sm' | 'md' | 'lg'
}) {
  const sizes = {
    sm: 64,
    md: 128,
    lg: 256,
  }

  return (
    <Image
      src={`/characters/miso/${emotion}.svg`}
      alt={`Miso the dolphin looking ${emotion}`}
      width={sizes[size]}
      height={sizes[size]}
      className="object-contain"
    />
  )
}
```

### Framer Motion Animation
```tsx
import { motion } from 'framer-motion'

export function FloatingMiso() {
  return (
    <motion.img
      src="/characters/miso/happy.svg"
      alt="Miso the dolphin"
      className="w-32 h-32"
      animate={{
        y: [0, -10, 0],
      }}
      transition={{
        duration: 3,
        repeat: Infinity,
        ease: "easeInOut"
      }}
    />
  )
}
```
